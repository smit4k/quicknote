# Quicknote  <img src="/src/main/resources/assets/quicknote/icon.png" align="right" width="40" height="40">
A lightweight Fabric mod that allows you to create and manage notes directly in your world through the chat.

## Installation
1. Install and setup [Fabric](https://fabricmc.net)
2. Download the `.jar` file from [Releases](https://github.com/smit4k/quicknote/releases)
3. Place the `.jar` file in your Minecraft `mods` folder
4. Launch Minecraft
